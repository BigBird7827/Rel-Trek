Rel-Trek

Rel-Trek is a Star Trek relationship explorer focused on people.

Choose a key character and see their important relationships.

Shows cards for each relationship. For each relationship, gives a short, canon-based overview of:

What the relationship is about
How it starts and changes
Key episodes and memorable moments

Covers TOS, TNG, DS9, VOY, ENT and their connected Prime-timeline films, using on-screen canon only.
