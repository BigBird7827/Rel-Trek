Rel-Trek

Rel-Trek is a Star Trek relationship explorer focused on people, not ships or tech.

Lets you pick a key character and see their important relationships.

Shows cards for each relationship (friends, rivals, family, lovers, political ties, etc.).

For each relationship, gives a short, canon-based overview of:

What the relationship is about

How it starts and changes

Key episodes and memorable moments

Covers TOS, TNG, DS9, VOY, ENT and their connected Prime-timeline films, using on-screen canon only.
