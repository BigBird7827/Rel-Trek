Rel-Trek

Rel-Trek is a Star Trek relationship explorer focused on people.
It lets you pick a key character and see their important relationships.
It shows cards for each relationship (friends, rivals, family, lovers, political ties, etc.).

For each relationship, it gives a short, canon-based overview of what the relationship is about, how it starts and changes, and key episodes or memorable moments.

It covers TOS, TNG, DS9, VOY and ENT, plus connected Prime-timeline films, using on-screen canon only.
